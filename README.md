# 📊 Excel Dashboard Projects

This repository contains two interactive Excel dashboard projects built to practice data analysis, data cleaning, and dashboard development using Microsoft Excel. The projects demonstrate the use of Power Query, Pivot Tables, Pivot Charts, Slicers, Timelines, Conditional Formatting, and KPI reporting to transform raw datasets into interactive business dashboards.

# 📁 Project 1 - California Fashion Sales Dashboard

## 📌 Overview

This dashboard analyzes a fashion retail dataset to provide insights into sales performance, customer purchasing behavior, product popularity, delivery efficiency, and regional distribution. Interactive filters allow users to explore the data dynamically.

### Dashboard Features

- Sales KPI Cards
- Weekly Sales Trend
- Customer Purchase Analysis
- Product Popularity by Gender
- Customer Location Analysis
- Delivery Time Distribution
- Customer Satisfaction Analysis
- Interactive Slicers

### Key Insights

- Identified the most popular product categories.
- Compared customer purchasing behavior across different shopping platforms.
- Analyzed average delivery time and customer satisfaction.
- Visualized customer distribution across different regions.
- Monitored weekly sales trends for performance tracking.

# 📁 Project 2 - Sales Analysis Dashboard

## 📌 Overview

This dashboard provides a detailed sales performance analysis using various business metrics such as revenue, customer spending, product categories, occasions, cities, and order trends. It helps understand sales patterns and customer purchasing behavior.

### Dashboard Features

- Revenue KPI Cards
- Revenue by Occasion
- Revenue by Category
- Revenue by Hour
- Monthly Revenue Trend
- Top Products by Revenue
- Top Cities by Orders
- Interactive Timeline & Filters

### Key Insights

- Identified occasions generating the highest revenue.
- Analyzed category-wise sales contribution.
- Determined peak ordering hours.
- Compared monthly revenue trends.
- Identified top-performing products and cities.
- Measured average customer spending and delivery time.
